# About
Projek ini merupakan implementasi sederhana Algoritma K-NN pada aplikasi Machine Learning untuk mengklasifikasikan Website. 
# Prerequiresite
```
python 3.X
pip
```

# Steps
## Clone Repository
```
git clone https://github.com/wildanisme/machine-learning-website-classification.git <optional directory>
```
## Enter Destination Directory
```
cd <directory>
```
## Setup Virtual Env
```
python3 -m venv .env
```
### Activate Virtual Env
```
source .env/bin/activate
```
### Install Required Packages
```
pip install -r requirements.txt
```
## Run Web
```
python3 app.py
```
